\begin{frame}[t]{Summary}
\textbf{Key Takeaways:}
\begin{itemize}
\item Markets are hard to predict (EMH partially true)
\item Regime changes invalidate historical patterns
\item Model decay inevitable: continuous monitoring required
\item Governance critical for production ML systems
\item Realistic expectations: Sharpe 1.5-2.0 is excellent
\item Alpha is scarce, crowded, and decays rapidly
\end{itemize}
